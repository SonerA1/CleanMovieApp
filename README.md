# CleanMovieApp
### CleanMovieApp 
We made our own movie application in MovieApp.
We sorted the movies with the search bar,
gave information about the movie with APIs, 
and took pictures.
In this application we made it using Clean Architecture.

## Used
-Room

-Hilt-Dagger

-Retrofit

-API

-Flow

-Full Compose

-Coroutine

-ViewModel

-Clean Architecture
